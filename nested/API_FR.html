Documentation API Beewant

Documentation complète pour l'intégration de l'API Beewant

Vue d'ensemble

L'API Beewant vous permet d'accéder programmatiquement à vos données de campagne. Cette API RESTful utilise JSON pour les requêtes et les réponses.

i
URL de base

https://api.beewant.com/v1

Authentification

Toutes les requêtes API nécessitent une clé API valide passée dans l'en-tête Authorization.

Authorization: Bearer YOUR_API_KEY
Note : Gardez votre clé API confidentielle et ne la partagez jamais publiquement.

GET /campaigns

Récupère une liste de toutes vos campagnes actives.

Paramètres de requête

Paramètre Type Description
limit integer Nombre de résultats à retourner (par défaut: 20, max: 100)
offset integer Nombre de résultats à ignorer (par défaut: 0)
status string Filtrer par statut : active, paused, completed

Exemple de requête

curl -X GET "https://api.beewant.com/v1/campaigns?limit=10&status=active" \
  -H "Authorization: Bearer YOUR_API_KEY"

Exemple de réponse

{
  "success": true,
  "data": [
    {
      "id": "camp_abc123",
      "name": "Campagne d'été 2024",
      "status": "active",
      "budget": 5000,
      "spent": 2340,
      "created_at": "2024-06-01T10:00:00Z"
    }
  ],
  "meta": {
    "total": 45,
    "limit": 10,
    "offset": 0
  }
}

GET /campaign/{id}

Récupère les détails d'une campagne spécifique.

Paramètres de chemin

Paramètre Type Description
id string L'identifiant unique de la campagne

Exemple de requête

curl -X GET "https://api.beewant.com/v1/campaign/camp_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"

Exemple de réponse

{
  "success": true,
  "data": {
    "id": "camp_abc123",
    "name": "Campagne d'été 2024",
    "status": "active",
    "budget": 5000,
    "spent": 2340,
    "impressions": 125000,
    "clicks": 3200,
    "conversions": 145,
    "created_at": "2024-06-01T10:00:00Z",
    "updated_at": "2024-06-15T14:30:00Z"
  }
}
Attention : Les données de campagne sont mises en cache pendant 5 minutes. Les modifications récentes peuvent ne pas apparaître immédiatement.
