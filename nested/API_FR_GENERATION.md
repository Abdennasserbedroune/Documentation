# API French page generation

This folder contains `nested/API EN.html` and helper scripts used to create French documentation pages. [cite:25]

## Create `API_FR.html`

Run:

```bash
python3 nested/create_api_fr.py
```

This generates:

- `nested/API_FR.html`

The generated file keeps the exact same design/structure and replaces visible text with professional French (targeted replacements to avoid breaking code snippets). [cite:29]
